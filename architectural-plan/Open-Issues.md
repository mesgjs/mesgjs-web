- Text Node Normalization Guards: The browser may automatically merge adjacent text nodes. If a reactive placeholder is a text node, it could be merged with a sibling and the VNode's reference to it would be lost. Reactive text content should instead be anchored to a non-normalizing "guard" node, such as a DOM comment node, to ensure the placeholder's reference is stable.
