This project is building the Mesgjs Web Interface (MWI), a bilingual
JavaScript-and-Mesgjs system for hybrid server-side and client-side
rendering of web interfaces from structured data.

The project will eventually need to include a foundational set of basic
components in order to be useful, but that will be the focus of a later
phase after the initial rendering system is in place.

Especially in early portions of the development, please look for opportunities
to break development into independently reviewable pieces. Check in often,
waiting for feedback before continuing, so that any necessary design revisions
can be caught early and without wasting a lot of resources.
